# DOCKER

### Les images Docker

  * Constituée d'une ou plusieurs **composants**  
  * Constituée d'une ou plusieurs **layers**  
       => Stockées dans **/var/lib/docker**
  * Distribuée via un Registry  
       => dockerHub

#### les composants

* Code _(java , ruby , python ...)_
* Dépendances _(jar...)_
* Runtime _(Node...)_
* OS _(Alpine...)_

_Chaque composant possède des **layers**_
